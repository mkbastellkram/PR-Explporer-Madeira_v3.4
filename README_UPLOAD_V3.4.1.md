# PR-Explorer V3.4.1 Austausch-Version

Diese Version korrigiert die Dateizuordnung.

Wichtig:
- `index.html` muss mit `<!doctype html>` beginnen.
- `app.js` muss mit `(() => {` beginnen.
- Lade den Inhalt dieses entpackten Ordners in deinen GitHub-Ordner hoch, nicht die ZIP-Datei selbst.

Empfohlener GitHub-Zielordner:

```text
PR-Explorer-Madeira_v3.4/
```

Darin müssen danach liegen:

```text
index.html
style.css
app.js
data.js
manifest.webmanifest
service-worker.js
assets/
```

Test-URL:

```text
https://mkbastellkram.github.io/PR-Explorer-Madeira_v3.4/index.html?v=3.4.1
```

Wenn Safari noch alten Inhalt zeigt: URL mit `?v=3.4.1` öffnen oder Cache/PWA neu laden.
