# PR Explorer Madeira · V3.3.4

UX-Korrekturstand auf Basis V3.3.3.

## Änderungen

- PR-Karussell lässt sich vertikal bedienen.
- Hochwischen auf Karussell/Griff öffnet das Detail-Sheet.
- Runterziehen blendet das Karussell aus.
- Detail-Schaltfläche auf den PR-Kacheln entfernt.
- Einstellungen sind wieder erreichbar: Topbar-Zahnrad und Bottom-Navigation „Einstellungen“.
- Einstellungs-Sheet mit Zugriff auf Karten/Layer, aktive PR einpassen, Gesamtkarte und Auswahl lösen.
- Karussell behält stabile Reihenfolge; aktiver PR wird nicht mehr an Position 1 umsortiert.
