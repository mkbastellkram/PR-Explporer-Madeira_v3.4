# PR Explorer Madeira · V3.3.2

Korrekturpaket nach iPhone-Test von V3.3.1.

## Änderungen

- Aktive PR zeigt GPX-Track und KML-Anfahrt automatisch, sobald sie ausgewählt ist.
- Karussell-Klick wechselt aktive PR, ohne die Karte erneut hart zu verschieben.
- Pin-Nummern erhalten höhere Priorität gegenüber POI-/Webcam-/Endpunkt-Markern.
- PR-Pins erhalten Statuspunkt unten rechts und Favoritenstern oben links.
- Safe-Area unten nochmals reduziert: Dock liegt näher am unteren Displayrand.
- Service-Worker-Cache auf V3.3.2 erhöht.
- OSM Hiking Layer bleibt im Layer-Menü enthalten.

## Hinweise

Der Status offen/eingeschränkt/geschlossen ist UI-seitig vorbereitet. Ohne belastbare Statusdaten im Datenbestand erscheint weiterhin ein neutraler/unklarer Status. Ein echter Live-Abruf muss separat gegen die Zielquelle geprüft werden.
