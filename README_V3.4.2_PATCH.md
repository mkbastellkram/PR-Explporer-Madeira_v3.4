- Reisezeitraum in Reise-Sammlung ergänzt.
- Filter-Reset „Alle PR“ ergänzt; „Alle“ setzt nun alle Filter zurück.
- Schutz gegen leere Karte durch versehentlich aktive Favorit-/Reise-/Statusfilter verbessert.

- POI-Basisdaten mit deutschen Kurzsteckbriefen und categoryId/baseIconId ergänzt.
